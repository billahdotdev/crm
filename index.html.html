⚡ Lead Engine v4 Setup

Google Sheets-কে database হিসেবে use করতে নিচের steps follow করুন।

1
Google Sheet তৈরি করুন — যেকোনো নাম দিন। Sheet URL থেকে Spreadsheet ID কপি করুন। URL: docs.google.com/spreadsheets/d/[THIS_PART]/edit
2
Apps Script Deploy করুন — Extensions → Apps Script এ গিয়ে নিচের apps_script.gs কোড paste করে Deploy করুন। "Execute as: Me", "Who has access: Anyone" সেট করুন। Deploy URL কপি করুন।
3
নিচে URL paste করুন এবং Save করুন।
